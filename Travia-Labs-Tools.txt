### Summary of Tool Endpoints (Including Higgsfield Native APIs):

*Note: Since Soul 2.0 and Speak V2 are NOT on fal.ai, these will be requested directly from Higgsfield's newly announced API using their Node.js SDK.*

| Tool                      | Higgsfield Model                           | Fal.ai Equivalent?                                   | Endpoint                                                                                                                           |
| ------------------------- | ------------------------------------------ | ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| UGC Factory / UGC Builder | Speech to Video [Speak V2]                 | Veo 3.1                                              | **Higgsfield API**: \`/speech-to-video\` OR **fal.ai**: \`fal-ai/veo3.1/reference-to-video\`                                           |
| Lipsync Studio            | WAN 2.5 + proprietary lipsync              | Kling LipSync + VEED Lipsync                         | **fal.ai**: \`fal-ai/kling-video/lipsync/...\` and \`veed/lipsync\`                                                                    |
| Soul ID (AI characters)   | Character [Soul ID]                        | Flux LoRA training + Nano Banana Pro                 | **Higgsfield API**: \`/character\` (Soul ID creation endpoint)                                                                         |
| High-Quality Image Gen    | Text to Image [Soul 2.0]                   | Imagen 4 (Nano Banana Pro)                           | **Higgsfield API**: \`/text-to-image\` OR **fal.ai**: \`fal-ai/nano-banana-pro\`                                                       |
| Video Gen (Fast)          | Image to Video [Dop]                       | Kling/Wan                                            | **Higgsfield API**: \`/image-to-video\`                                                                                                |
| Click to Ad               | Combo: Speak V2 + scripting + Soul ID      | Can replicate with multiple models                   | *Pipeline built on backend utilizing Higgsfield endpoints*                                                                         |
